# 📘 Prática com Db2 em Nuvem

Hoje realizei um exercício utilizando o **Db2 em nuvem**, colocando em prática alguns conceitos estudados sobre consultas e administração de banco de dados.

## ✅ O que pratiquei

- Conexão com o ambiente Db2 Cloud
- Utilização do comando `SELECT`
- Consulta de dados em tabelas
- Exportação de informações para um arquivo `.csv`
- Observação da importância dos logs para recuperação e rastreamento de operações

## 🧠 Aprendizado

Esse exercício ajudou a reforçar como os dados podem ser consultados e exportados para análise externa, além de mostrar como os registros de transação são importantes para manter a integridade e a recuperação das informações em caso de falhas.
