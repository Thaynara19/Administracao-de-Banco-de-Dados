## 🧠 O que aprendi

Entendi que os bancos de dados relacionais possuem uma **organização hierárquica**, com diferentes níveis dentro do servidor.

A estrutura geral funciona assim:

Instância → Banco de Dados → Esquemas → Tabelas → Índices

A **instância** representa o ambiente principal onde o banco está em execução.

Também aprendi conceitos importantes:

- Metadados são informações que descrevem o próprio banco de dados, como permissões e estrutura
- O sistema mantém dados internos para controle e funcionamento
- Tablespaces facilitam o gerenciamento do armazenamento sem precisar conhecer o local físico dos arquivos
