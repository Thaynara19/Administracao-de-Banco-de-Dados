# 📘 Administração de Banco de Dados Relacional (DBA) — Objetos e Hierarquia do Servidor

## ✅ O que estudei hoje

Hoje avancei para a parte de **objetos e hierarquia do servidor de banco de dados**.

Comecei a entender melhor que o banco de dados não funciona de forma isolada. Existe uma estrutura organizada dentro do servidor que controla como os bancos, tabelas e configurações são gerenciados.

Entre os conteúdos estudados:

- Objetos de banco de dados
- Objetos do sistema e configurações do banco
- Armazenamento de banco de dados
- Introdução aos mecanismos de armazenamento no MySQL
- Preparação para laboratórios práticos com MySQL e PostgreSQL

## 🧠 Entendimento pessoal

Antes eu pensava que o banco de dados era apenas composto por tabelas.  
Agora entendo que existe uma **hierarquia**, onde o servidor administra vários componentes responsáveis pelo funcionamento, organização e armazenamento dos dados.

Percebi que o DBA precisa entender não só os dados, mas também **como o banco é estruturado internamente**, incluindo configurações do sistema e formas de armazenamento.

## 🚀 Evolução no aprendizado

Sinto que estou começando a enxergar o banco de dados de uma forma mais profissional, entendendo que administrar dados envolve infraestrutura, organização e monitoramento contínuo do ambiente.
